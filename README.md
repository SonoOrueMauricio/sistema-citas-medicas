# Sistema de Gestión de Citas Médicas

Proyecto desarrollado para el curso Integrador I – Sistemas Software.

## Descripción

Este sistema permite gestionar citas médicas de manera eficiente, facilitando el registro de pacientes, programación de citas y visualización de la información en tiempo real.

## Tecnologías utilizadas

- Python (Flask)
- PostgreSQL
- HTML, CSS, JavaScript
- Git y GitHub

## Funcionalidades

- Sistema de login de usuarios
- Registro de pacientes
- Programación de citas médicas
- Visualización de citas por usuario
- Calendario interactivo de citas
- Gestión de especialidades médicas

## Estructura del proyecto

- Backend: Flask (Python)
- Frontend: HTML, CSS, JavaScript
- Base de datos: PostgreSQL
